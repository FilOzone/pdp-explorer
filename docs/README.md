# PDP Explorer Documentation

This directory documents the two live components of the PDP Explorer:

- **`subgraph/`** — the PDP subgraph, indexed and served via [Goldsky](https://goldsky.com).
- **`subgraph-client/`** — the web app that queries the subgraph.

## Subgraph

- [**Development**](subgraph/development.md): Configuring, building, testing, and deploying development versions of the subgraph.
- [**Release process**](subgraph/release.md): Source of truth for versioned deployment, indexing, verification, Goldsky `prod` promotion, and rollback.
- [**Release issue template**](../.github/ISSUE_TEMPLATE/subgraph_release.md): Per-release progress tracking generated by the release workflow.
- [**GraphQL API**](subgraph/graphql-api.md): Schema, entities, and example queries exposed by the subgraph.
